2. Two-Way Binding - [(ngModel)]

Synchronizes data between component and template (both directions)

Syntax: [(ngModel)]="propertyName"

User Name: {{ userName || 'Not entered yet' }}

User Email: {{ userEmail || 'Not entered yet' }}

Greeting: {{ greeting }}

3. Property Binding - [property]

Binds component properties to DOM element properties

Syntax: [property]="expression"

4. Event Binding - (event)

Responds to user events like click, input, etc.

Syntax: (event)="handler()"

Button Clicked: {{ clickCount }} times

5. Structural Directives - *ngIf

Syntax: *ngIf="condition"
