❓ How Do You Pass $event in Angular?

1. Passing Native DOM Event Object

<input (input)="onInput($event)">

// In component:
onInput(event: Event): void {{ '{' }}
  const value = (event.target as HTMLInputElement).value;
{{ '}' }}

2. Passing Custom Data

<button (click)="onProductClick(product)">
  Select
</button>

// In component:
onProductClick(product: Product): void {{ '{' }}
  this.selectedProduct = product;
{{ '}' }}

3. Passing Both $event and Custom Data

<button (click)="onData($event, productId)">
  Click
</button>

// In component:
onData(event: Event, id: number): void {{ '{' }}
  console.log('Event:', event);
  console.log('ID:', id);
{{ '}' }}

4. Common $event Properties

Property Usage Example
event.target Get the element that triggered event (event.target as HTMLInputElement).value
event.type Get event type (click, input, etc) event.type
event.preventDefault() Prevent default behavior (keyup.enter)="handler($event)"
event.stopPropagation() Stop event bubbling event.stopPropagation()
