❓ Difference Between [style.background] vs [ngStyle]

Aspect [style.property] [ngStyle]
Syntax [style.backgroundColor]="color" [ngStyle]="{{ '{' }} 'background-color': color {{ '}' }}"
Single Style ✅ Simple & Clean ⚠️ Verbose
Multiple Styles ⚠️ Multiple bindings ✅ Single binding
Readability ✅ Clear for one/two styles ✅ Clear for many styles
Performance ✅ Slightly better Comparable

Single Style Example

[style.backgroundColor]:
<div [style.backgroundColor]="bgColor"></div>

[ngStyle]:
<div [ngStyle]="{{ '{' }} 'background-color': bgColor {{ '}' }}"></div>

Multiple Styles Example

Multiple [style]:
<div [style.backgroundColor]="bgColor"
      [style.color]="textColor"
      [style.fontSize.px]="fontSize"></div>

[ngStyle] (Cleaner):
<div [ngStyle]="{{ '{' }} 'background-color': bgColor, 'color': textColor, 'font-size': fontSize + 'px' {{ '}' }}"></div>
