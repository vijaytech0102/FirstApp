Combining Interpolation & ngModel

📊 Mark Based Pass/Fail Checker

Marks: {{ marks !== null ? marks : '(not entered)' }}

Result: {{ result }}

🎯 Passing Grade Criteria

✅ Pass: Marks ≥ 40

❌ Fail: Marks < 40

❓ Can We Use Ternary Operator Inside Interpolation?

✅ YES! We can use ternary operator inside interpolation

{{ condition ? valueIfTrue : valueIfFalse }}

Example: {{ marks !== null ? (marks >= 40 ? 'Pass' : 'Fail') : 'Not entered' }}

This component uses ternary in component method, but we can also write directly in template
