📝 How Angular Handles DOM Events

Event Binding Syntax: (event)="handler()"

1. Event Registration

Angular registers the event listener on the DOM element

2. Event Occurrence

User interacts (click, input, etc.), browser triggers the event

3. Zone.js (Change Detection)

Zone.js intercepts the event and notifies Angular's change detection

4. Handler Execution

Angular calls the method/handler from component class

5. Change Detection

Angular runs change detection, updates bindings and DOM

Common DOM Events

(click) Mouse click or Enter key on button
(input) Text input change
(change) Element loses focus after change
(submit) Form submission
(keyup) Key released
(keydown) Key pressed
(mouseover) Mouse moves over element
💡 Tip: Open the browser console (F12) to see console.log output when you click the button!
