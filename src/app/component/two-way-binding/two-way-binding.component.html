How ngModel Works?

Syntax: [(ngModel)]="propertyName"

✓ Combines property binding [ngModel] and event binding (ngModelChange)

✓ Creates two-way synchronization between input field and component property

✓ Changes in input → updates component property

✓ Changes in property → updates input field

❓ Which Module Must Be Imported?

✅ FormsModule must be imported to use ngModel

import {{ '{' }} FormsModule {{ '}' }} from '@angular/forms';

// In component standalone or NgModule:
imports: {{ '[' }}FormsModule{{ ']' }}
