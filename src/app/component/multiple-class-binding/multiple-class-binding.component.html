❓ Different Ways to Use ngClass

1. Object Syntax (Most Common)

[ngClass]="{{ '{' }} active: isActive, disabled: isDisabled {{ '}' }}"

Key-value pairs where:
• Key = CSS class name
• Value = boolean condition

2. Array Syntax

[ngClass]="{{ '[' }}'class1', 'class2'{{ ']' }}"

Array of class names to always apply
[ngClass]="{{ '[' }}'class1', condition ? 'class2' : 'class3'{{ ']' }}"

Can use ternary operators inside

3. String Syntax

[ngClass]="dynamicClassName"

Single class name from a variable

4. Combination - Object + Array

[ngClass]="{{ '{' }} active: isActive, ...otherClasses {{ '}' }}"

Merge multiple sources in object syntax
Syntax Use Case Example
Object Multiple conditional classes [ngClass]="{{ '{' }} active: isActive, error: hasError {{ '}' }}"
Array Mix of static and dynamic classes [ngClass]="{{ '[' }}'base', 'theme'{{ ']' }}"
String Single dynamic class [ngClass]="dynamicClass"
[class.name] Single class binding [class.active]="isActive"
