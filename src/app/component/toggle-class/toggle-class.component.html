Step 2: Create a Toggle Method

🔄 Other Toggle Patterns

Pattern 1: Simple Boolean Toggle
toggle(): void {{ '{' }}
  this.isVisible = !this.isVisible;
{{ '}' }}
Pattern 2: Toggle with Logging
toggle(): void {{ '{' }}
  this.isVisible = !this.isVisible;
  console.log('Toggled:', this.isVisible);
{{ '}' }}
Pattern 3: Toggle with Callback
toggle(callback?: () => void): void {{ '{' }}
  this.isVisible = !this.isVisible;
  callback?.();
{{ '}' }}
