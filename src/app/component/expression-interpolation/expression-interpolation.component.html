❓ Can We Use Complex Logic (Loops/Conditions)?

❌ No, we cannot use:

  • Loops (for, while, etc.)
  • If-else statements directly
  • Assignment operators (=, +=, etc.)

✅ We can use:

  • Method calls that contain logic
  • Ternary operator: (condition ? true : false)
  • Optional chaining operator: variable?.property
  • Simple arithmetic and string operations
